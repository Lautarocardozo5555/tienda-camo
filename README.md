# Tienda CAMO

Sitio web desarrollado como parte de la Preentrega 2 del curso de Desarrollo Web en Coderhouse. CAMO es una tienda de ropa con estilo minimalista, moderno y funcional, orientada a ofrecer una experiencia digital accesible y responsive.

## Descripción

Este proyecto incluye múltiples páginas con estructura semántica, diseño responsive y buenas prácticas de accesibilidad. El sitio está publicado mediante GitHub Pages y versionado con Git.

## Tecnologías utilizadas

- HTML5 semántico  
- CSS3 con variables, pseudoclases y media queries  
- Flexbox y CSS Grid  
- Bootstrap 5  
- Git y GitHub  
- GitHub Pages

## Diseño responsive

El sitio se adapta a distintos tamaños de pantalla mediante media queries y clases de Bootstrap. Se optimizó para móviles, tablets y escritorio, evitando scroll horizontal y manteniendo buena legibilidad.

## Funcionalidades destacadas

- Uso de `<h1>` único por página para mejorar accesibilidad y SEO  
- Navegación clara entre secciones  
- Cards, grillas y formularios estilizados con Bootstrap  
- Estructura modular con CSS organizado  
- Control de versiones con commits descriptivos
